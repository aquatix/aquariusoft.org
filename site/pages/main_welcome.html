News

Currently the most active projects are the web projects. However, the aquamorph livecd is still under active development to get a polished livecd for general use. Also the other projects aren't forgotten either. I just have to spread my sparse free time among the things that interest me most at a given time and I don't want to spread it thin.

aquamorph

Take a look at a work in progress: the aquamorph livecd. This Morphix and Debian GNU/Linux based livecd aims to be easy to use, pleasing to the eye, up-to-date and just generally being usable and userfriendly.

The Qik site framework

One of the projects of AquariuSoft is Qik, a PHP based framework for quickly setting up a site, even multilingual if you want. No need to use databases, only PHP has to be installed. You don't even have to know anything about PHP programming to set up a site with it!

windows programs

The old Windows programs aren't under active development anymore; detailed product pages are still missing, but there's an overview page, from which the [rather old] versions can be downloaded and a short description can be read. Mind that contact information mentioned in the programs is outdated.
