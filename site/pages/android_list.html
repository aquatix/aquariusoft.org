NB: I'm not endorsed by any of these applications; they are just being extremely handy, great to have or otherwise amusing. See it as a repository or treasurebox for your Android phone.

This page is being constructed; a short description will be added to the items, as will a link to the respective Androlib pages and a QR-code.

Start with getting Barcode Scanner and find the respective QR-code's of the applications on Androlib. Great source of applications BTW.

Games

  • Andoku - sudoku game
  • Armadillo Roll - 3D racing game rolling an armadillo down a track
  • Bonsai Blast
  • Flood-It!
  • Frodo - Commodore64 emulator
  • Graviturn
  • MarioSimulator - bounce around to the sound of Mario jumps and coins
  • Pong - Pong!

Location based info

  • Here I Am - find and send your current location
  • Layar - show information about the area surrounding you; many layers available, like house prices, bars etc
  • Wikitude - similar augmented reality application as Layar, but by the community

Networking, communication

  • ConnectBot - ssh client
  • DroidIn-Lite - LinkedIn client
  • Hyves Mobile (soon) - Hyves client
  • Meebo IM - Instant Messaging client
  • wpToGo - Edit Wordpress weblog from your phone

News, information, weather etc.

  • Het Weer - great Dutch weather app, including Buienradar images and more
  • NewsRob - (offline) google (rss)reader client
  • NU - nu.nl client, which also has tv guide etc
  • TED
  • NOS Teletekst - Dutch tv teletext application
  • Wapedia

Productivity

  • Astrid - todo list which can sync with Remember the Milk
  • Aurora - global search: find something in all information available on the device, from sms and mail to web history and media
  • ColorDict - dictionaries, from thesauri to wikipedia addon
  • Detexify - write the character on the screen and let it find out the TeX code
  • Notes - note taking application
  • Share Your Board - capture a whiteboard with the camera and make it into a readable image (de-warp, enhance contrast etc)

Phone admin

  • Astro - great file manager with build-in task manager, zip support and more
  • Linda File Manager
  • Locale - change settings and do other things based on time, location and more
  • NetMeter
  • NetSentry
  • Phonalyzr - show information about the calling history
  • Spare Parts - extra settings
  • Task Manager - manage open applications and processes
  • Xtremelabs Speedtest - test internet connection speed

Shopping

  • Pricewatch Scanner - scan barcode of technical item and compare prices in the Tweakers pricewatch
  • SnapTell

Tools

  • Barcode Scanner - scan 1D and 2D barcodes and qr-codes
  • BeeTagg - scan 1D and 2D barcodes and qr-codes
  • Bubble - spirit level (NL: waterpas)
  • FBReader - ebook reader
  • handyCalc - super complete and really great designed calculator and equation solver. Even does unit and currency conversions
  • OI Safe - password manager
  • SDrescan - force rescan of the SD card (helps against mediaplayers and such not getting update metadata)
  • Shazam - recognise music playing around you

Toys

  • Comic viewer - read comics in CBZ/ZIP, CBR/RAR (beta), ACV, JPEG, PNG, BMP, GIF files

  • gStrings - chromatic tuner application measuring sound pitch and intensity
  • Metal Detector
  • Ringdroid

Tracking and GPS various

  • Compass - compass application using the electronic compass in the phone
  • Google Sky Map - show the stars and find other celestial bodies
  • GPS Speedo - speedometer using gps information
  • Location Log

  • Places Directory - allows you to browse nearby places in categories like Restaurants, Movie Theaters and Hotels
