HTML modules package

When developing websites and web applications, you come to build certain functionality over and over again. To make building an application faster, it would be nice to have a library of existing functions to use instead of reinventing the wheel for the zillionth time. That was the reason I'm currently collecting all kinds of useful PHP functions into modules, which can be used in web applications and will be put here to be downloaded for Free [GPL'ed].
