Qik site framework

Framework to quickly set up a website from scratch. Use flat files or a database, your choice.

The current version, which will be available online in a few days, uses flat files to quickly build a full-fletched website, like the aquariusoft.org website you're currently visiting, of the website of the Dutch basketball club Mapleleaves.
