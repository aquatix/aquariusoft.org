overload feedreader

De overload feedreader is een web-gebaseerd, Open Source [uitgebracht onder de GPL] nieuws feed lezer, welke kan worden gebruikt op zijn eigen adres, of kan worden gedownload en geinstalleerd op uw eigen webserver. Hiervoor is alleen PHP en een MySQL database benodigd.

Op dit moment wordt het project herschreven in het Django framework, wat gebaseerd is op Python. Het zal een "informatiemanager" worden, waarin ook uw bookmarks, notities etc kan worden opgeslagen. De gebruikersinterface wordt opnieuw ontworpen en zal gebruik maken van tags om informatie op een overzichtelijke manier op te slaan.
